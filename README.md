## üöÄ Project Overview
In today's e-commerce market, consumers often face "choice paralysis" due to scattered product information and rapidly changing prices across platforms. Our Intelligent Shopping Agent consolidates data from multiple sources to help users make informed, cost-effective decisions efficiently.

### Key Features

**Real-time Price Transparency:** Scrapes and compares prices across platforms to ensure users find the lowest cost.
* 


**Personalized Suggestions:** Learns user preferences and budget to offer tailored product recommendations
* 


**Granular Accuracy:** Utilizes an Intent Parser to avoid generic search results and focus on specific user needs.
* 


**Structured Advice:** Converts messy web data into clean, readable shopping advice through specialized extraction agents.

## üèóÔ∏è System Architecture

The agent operates through a sophisticated node-based workflow that manages the journey from user intent to the final recommendation.

### Agentic Workflow

1. 
**Intent Understanding Agent:** Parses the user's natural language query.

2. 
**Product Discovery Agent:** Scours various e-commerce sites using real-time search.

3. 
**Pricing & Availability Agent:** Validates current costs and stock status.

4. 
**Unified Recommendation Agent:** Ranks products based on the processed data and user constraints.
